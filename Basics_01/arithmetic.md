### **1. Arithmetic Operators**

Used to perform mathematical calculations.

| Operator | Description | Example |
| --- | --- | --- |
| `+` | Addition | `5 + 2` → `7` |
| `-` | Subtraction | `5 - 2` → `3` |
| `*` | Multiplication | `5 * 2` → `10` |
| `/` | Division | `5 / 2` → `2.5` |
| `%` | Modulus (Remainder) | `5 % 2` → `1` |
| `**` | Exponentiation | `2 ** 3` → `8` |
